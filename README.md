# Shopify Testing Applications

## Folder Structure

- client
- server

## Client

### Installation

```sh
yarn install
```

### Run

```sh
yarn dev
```

## Server

### Installation

```sh
yarn install
```

### Run

```sh
yarn start:dev
```
